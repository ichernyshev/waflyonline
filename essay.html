Эссе на тему: "Почему GitHub, а не другой хостинг?"

Существует множество разных бесплатных хостингов, но у GitHub есть ряд преимуществ перед ними:

Во-первых, большинство сторонних сервисов, полезных при разработке (ci серверы, анализаторы кода, багтрекеры, agile доски), работают только с GitHub'ом. Одно это уже является критическим доводом использования GitHub.

Во-вторых, всё современное девелоперское комьюнити сидит на GitHub. На GitHub все были, им все пользуются, он всем привычен. Поэтому вполне естественно именно его выбирать как хостинг для своего кода.

В-третьих, активность на GitHub может выступать в роли вашего резюме. Если вы активный контрибьютор OpenSource проектов, это может очень сильно помочь при смене места работы.

В четвёртых, на GitHub есть целая куча разных мелких удобных вещей, которых нет на его аналогах. В него вкладывают много денег, его активно развивают, поэтому в нём больше различных специфических способностей.
